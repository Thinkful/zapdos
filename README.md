# Usage

<!-- usage -->

```sh-session
$ npm install -g @thinkful/zapdos
$ zapdos COMMAND
running command...
$ zapdos (-v|--version|version)
@thinkful/zapdos/0.0.0 darwin-x64 node-v8.12.0
$ zapdos --help [COMMAND]
USAGE
  $ zapdos COMMAND
...
```
